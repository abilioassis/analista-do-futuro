**ÉPICO: Implementar Camada de Benefícios VIP**

**Narrativa do Épico:** Como Cliente VIP (High Value), Quero ser reconhecido automaticamente no checkout e receber descontos exclusivos no frete, Para que eu me sinta valorizado e opte por comprar aqui em vez da concorrência durante a Black Friday.

**Critérios de Aceitação:**
- O checkout deve consultar o serviço de CRM/Clusterização para identificar se o usuário logado possui a flag/tag "VIP".
- Caso o usuário seja VIP, o sistema deve aplicar uma regra de desconto (ex: Frete Grátis ou 50% OFF) que sobreponha as regras restritivas de custo do Épico 1.
- A interface (UI) do checkout deve exibir uma mensagem visual clara (ex: "Desconto VIP aplicado"), destacando o benefício financeiro para o usuário.

**SUA MISSÃO:** Crie uma descrição de Wireframe de Baixa Fidelidade (apenas texto) para a nova tela de Carrinho.

**ESTRUTURA DA RESPOSTA:**
1. **Elementos de Informação:** Liste os dados obrigatórios (ex: Valor original vs Valor com desconto).
2. **Hierarquia Visual:** O que deve ter mais destaque? O que é secundário?
3. **Microcopy (Texto de Interface):** Crie frases para dois estados:
   - Estado A (Sucesso): O cliente atingiu a meta (Carrinho cheio).
   - Estado B (Upsell): O cliente ainda não atingiu o valor (Incentivo).

**CENÁRIOS DE USUÁRIO:** Considere um Cliente VIP navegando na Black Friday.
